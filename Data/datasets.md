# Datasets

This project uses publicly available datasets for fraud detection research.

## Credit Card Fraud Dataset

Source:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Contains anonymized European card transactions used for machine learning fraud detection research.

## PaySim Dataset

Source:
https://www.kaggle.com/datasets/ealaxi/paysim1

A simulated mobile money transaction dataset modeling financial transactions in emerging economies.
